# Chloe GitBook Demo Content

This is a dummy GitBook-compatible docs repository.

## How to use

1. Create a new GitHub repository, for example `chloe-gitbook-demo-content`.
2. Upload or push these files.
3. In GitBook, connect a test space to the repository using Git Sync.
4. Use the pages to test editing, syncing, change requests, AI Assistant, permissions, and site customization.

## Good test scenarios

- Edit `README.md` in GitBook, sync, and check the GitHub commit.
- Edit `getting-started/quickstart.md` in GitHub, commit to main, and check GitBook updates.
- Create a branch, change `api/example-endpoint.md`, open a pull request, merge it, and check GitBook.
- Ask the AI Assistant: "How do I create a sandbox account?" and "What do I do if my API key does not work?"
- Hide or restrict `internal/release-checklist.md` and test audience controls.
